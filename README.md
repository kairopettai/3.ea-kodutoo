## Laiendus
Laienduse eesmärk on vahetada kõik pildid veebilehel ära minu pildiga. Isegi töötab videode thumbnailidega. Kuid mingipärast näiteks kui google otsingus piltide asemel panna minu pilt, siis alla kerides ja uusi pilte genereerides, enam uusi pilte ümber ei muuda.
